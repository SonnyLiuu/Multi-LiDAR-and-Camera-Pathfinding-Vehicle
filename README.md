A Wi-Fi-connected mobile robot that autonomously follows a colored target. A camera (ESP32-CAM) streams live MJPEG video to a host that runs a real-time vision pipeline (OpenCV + NumPy). The pipeline thresholds the image in HSV (with hue wrap), filters noise by keeping the largest connected component, computes the blob centroid, and issues movement commands (W/A/D/S) based on blob position.
